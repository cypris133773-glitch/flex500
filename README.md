# Flex500 Fitness App

## Overview
Flex500 is a professional fitness application designed to enhance the fitness experience with personalized features and a modern dashboard.

## Features
- **Personalized Workout Generator**: Generate workouts tailored to individual fitness levels and goals.
- **Progress Tracking with Charts**: Visualize your fitness journey with engaging charts and statistics.
- **Workout History**: Log and access past workouts easily.
- **Analytics**: Dive into detailed analysis of your fitness performance.
- **Multiple Trackers**: Keep track of various fitness metrics.
- **Goal Setting**: Set and manage personal fitness goals.
- **Saved Plans Management**: Save and manage workout plans for easy access.
- **Exercise Database**: Comprehensive database with 500+ exercises.
- **Intelligent Split Selection**: Choose optimal workout splits based on user goals.
- **Workout Player with Rest Timers**: Integrated player for guided workouts with timers for rest intervals.
- **Polished iOS-style UI**: Clean and modern user interface inspired by iOS design principles, featuring a blue and white theme.

## Main Dashboard
The main dashboard will include:
- Daily progress
- Personalized workout recommendations
- Key statistics
- Quick access to start workouts with in-depth tracking.
